# 양지웅 - 이력서 & 포트폴리오

GitHub Pages로 호스팅되는 웹 이력서와 프로젝트 포트폴리오입니다.

## 바로가기

- **이력서**: https://peppertunacan.github.io/Resume/
- **포트폴리오**: https://peppertunacan.github.io/Resume/portfolio.html

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 이력서 — 약력, 기술 스택, 프로젝트 요약, 경력/학력, 자격증 |
| `portfolio.html` | 포트폴리오 — 프로젝트별 Problem-Solution-Result 상세 기술 |
| `screenshots/` | 포트폴리오에 사용되는 프로젝트 스크린샷 |
